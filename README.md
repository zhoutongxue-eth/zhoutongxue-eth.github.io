# 区块链周同学主页

本主页使用hugo搭建.

## hugo

### 本地启动服务

```shell
hugo server
```

### 生成静态网页

```shell
hugo
```

## 主题

https://github.com/rhazdon/hugo-theme-hello-friend-ng

### 下载主题到本地

```shell
git submodule update --init
```

## 工具

[生成icon](https://realfavicongenerator.net/)

```shell
git config --local user.name test

```
